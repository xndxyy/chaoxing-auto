# 超星学习通 智能刷课助手 — 零基础使用教程

## 你需要的东西

- 一台 Windows 电脑
- 能上网
- 学习通账号（手机号+密码 或 学习通APP扫码）

---

## 第一步：下载本项目

### 方式A：从 GitHub 下载（推荐）
1. 打开项目页面
2. 点击绿色 **Code** 按钮 → **Download ZIP**
3. 解压到桌面（或任意位置）

### 方式B：直接复制
把整个文件夹复制到你的电脑上即可。

---

## 第二步：安装环境

双击文件夹里的 **`安装.bat`**

它会自动帮你：
- 检查 Python（如果没有，会打开下载页面）
- 安装浏览器引擎
- 安装 AI 答题模型

```

║          超星学习通 智能刷课助手 - 环境安装          ║
╚══════════════════════════════════════════════════════╝

[1/5] 检查 Python...
  ✅ Python 3.12.0
[2/5] 安装 Python 依赖...
  ✅ Python 依赖已安装
[3/5] 安装 Chromium 浏览器引擎...
  ✅ Chromium 已安装
[4/5] 检查 Node.js...
  ✅ Node.js v20.x.x
[5/5] 安装 OpenCode CLI...
  ✅ OpenCode 1.15.13

  ✅ 安装完成！
```

### 如果提示"未检测到 Python"

1. 安装脚本会自动打开 Python 下载页面
2. 点击黄色按钮下载
3. 运行安装程序
4. **重要：勾选 "Add Python to PATH"**（安装界面最下面的勾选框）
5. 安装完成后，重新双击 `安装.bat`

### 如果提示"未检测到 Node.js"

Node.js 只是 AI 答题需要，不装也能刷视频。如果需要：
1. 访问 https://nodejs.org
2. 下载 LTS 版本安装即可

### 正常流程

1. **输入 `1` 回车** → 开始刷课
2. **浏览器自动弹出** → 显示学习通登录页
3. **用手机学习通APP扫码** → 登录成功
4. **回到黑色终端窗口，按回车**
5. **显示课程列表** → 输入课程编号，比如 `3`
6. **如果提示"手动点击第一个章节"** → 在浏览器里点一下第一个视频章节，回来按回车
7. **自动开始刷课** → 全自动，不用管了

## 想调整设置？

在主菜单里选对应编号：

| 你想 | 操作 |
|------|------|
| 调快/调慢速度 | 选 `4` → 选择速度 |
| 开启秒过模式 | 选 `4` → 选 `6`（秒过） |
| 关闭AI答题 | 选 `5` → 选 `2` |
| 填写账号密码 | 选 `6` → 选 `1` |
| 测试AI是否正常 | 选 `7` |

**不需要打开任何代码文件！**

## 常见问题

### Q: 视频刷完了但显示未完成？
开启"防回滚"功能（默认已开启）。如果还不行，降低速度到 4x 试试。

### Q: AI答题不工作？
在主菜单选 `7` 测试AI。如果失败，检查是否安装了 Node.js 和 OpenCode。

### Q: 浏览器没弹出来？
关掉重试。如果还不行，在主菜单选 `8` 检查依赖是否全部 ✅。

### Q: 提示"请手动点击第一个章节"？
在弹出的浏览器里，点击课程的第一个章节/视频，然后回到终端按回车。

### Q: 想中途停止？
在终端按 `Ctrl + C`。

### Q: 换一门课刷？
一门课刷完后会自动询问是否继续。或者按 `Ctrl + C` 停止后重新运行。
